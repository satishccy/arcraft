Arcraft

JavaScript SDK for ARC's utils in Algorand for Node.js backends.

Description

This package provides utility functions for working with Algorand's ARC standards in Node.js environments. It's designed specifically for backend applications and servers.

Installation

npm install arcraft

Usage

import { createARC3Asset } from 'arcraft';

// Use the SDK functions

Pinata File Upload Examples

You can upload files to Pinata/IPFS in two ways:

1. Using a file path:

import { uploadToPinata } from 'arcraft';
import path from 'path';

async function uploadAssetFromFile() {
  try {
    const result = await uploadToPinata({
      file: path.resolve('./assets/myImage.jpg'),
      name: 'myImage.jpg',
      token: 'YOUR_PINATA_API_TOKEN',
    });
    
    console.log('Uploaded to IPFS with hash:', result.IpfsHash);
  } catch (error) {
    console.error('Upload failed:', error);
  }
}

2. Using a Buffer:

import { uploadToPinata } from 'arcraft';
import fs from 'fs';

async function uploadAssetFromBuffer() {
  try {
    // Read file into buffer
    const fileBuffer = fs.readFileSync('./assets/myImage.jpg');
    
    const result = await uploadToPinata({
      file: fileBuffer,
      name: 'myImage.jpg',
      token: 'YOUR_PINATA_API_TOKEN',
    });
    
    console.log('Uploaded to IPFS with hash:', result.IpfsHash);
  } catch (error) {
    console.error('Upload failed:', error);
  }
}

3. Uploading JSON:

import { uploadJsonToPinata } from 'arcraft';

async function uploadMetadata() {
  const metadata = {
    name: "My NFT",
    description: "This is my NFT description",
    image: "ipfs://QmYourImageCidHere",
    properties: {
      // Your properties here
    }
  };
  
  try {
    const result = await uploadJsonToPinata({
      json: metadata,
      name: 'metadata.json',
      token: 'YOUR_PINATA_API_TOKEN',
    });
    
    console.log('Metadata uploaded to IPFS with hash:', result.IpfsHash);
  } catch (error) {
    console.error('Upload failed:', error);
  }
}
